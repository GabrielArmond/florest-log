# Teste Frontend V4

## Setup

Instale as dependências

```bash
# npm
npm install
```
## Development Server

Iniciar no servidor de desenvolvimento `http://localhost:3000`:

```bash
# npm
npm run dev
```

## Production - deploy

Comando para buildar aplicação para deploy:

```bash
# npm
npm run build
```

Preview do build de produção

```bash
# npm
npm run preview
```
